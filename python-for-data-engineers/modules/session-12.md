# Session 12 — Capstone: End-to-End Pipeline

**Duration:** 5 hours | **Project:** [Capstone brief](../labs/starter/lab-12-capstone.md)

## Objectives

- Integrate all course skills into one pipeline
- Deliver documented, testable, rerunnable code
- Present design decisions to peers

## Capstone Requirements

### Must have

1. **Extract** from CSV + JSON + mock API files
2. **Transform** — clean, join, aggregate
3. **Load** — Parquet and/or SQLite tables
4. **Config** — YAML or `.env`
5. **Tests** — minimum 3 pytest cases
6. **Logging** — INFO-level run log
7. **README** — how to run

### Nice to have

- Data quality report
- CLI with date parameter
- Incremental load watermark

## Presentation Format (15 min each)

1. Architecture diagram (2 min)
2. Live demo (8 min)
3. What you'd do differently in production (5 min)
